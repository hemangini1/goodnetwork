A) Included Files - top

vc_row_background_particles folder represents the plugin itself and contains all necessary files for this plugin:

  1. img, scripts folder
  2. vcrbp.css
  3. vc_row_background_particles.php
  4. vc_column_background_particles.php

B) Installation - top

  1. Install the plugin .zip from the Plugins -> Add New -> Upload
  2. Activate the plugin through the 'Plugins' menu in WordPress
  3. Visit Visual Composer editor to config for row elements.

C) How to use - top

After VC Row Background Particles is installed and activated, you should see Background Particles tab when you edit settings for Row element:

  1. Select Background Style Effect
  2. Custom options
  3. Click Save Changes.

List Effect:

  1. Lightweight Particle
  2. Lightweight Dust Particle
  3. Lightweight Star Particle
  4. Lightweight Snow Particle
  5. ParticleGround
  6. Plasma Particle
  7. Organic Lines Particle
  8. Rainbow Curves Particle
  9. Rainbow Fireflies Particle
  10. Rainbow Grid Particle
  11. Centaurus
  12. Gradient Connection
  13. Shocking Connection
  14. Twinkle
  15. Rain Particle (Deprecated)
